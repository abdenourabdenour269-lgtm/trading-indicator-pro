## 🚀 المميزات الرئيسية

### 📈 المتوسطات المتحركة (Moving Averages)
- **SMA 20** - المتوسط البسيط لـ 20 فترة
- **SMA 50** - المتوسط البسيط لـ 50 فترة
- **EMA 12** - المتوسط الأسي لـ 12 فترة
- **EMA 26** - المتوسط الأسي لـ 26 فترة
- إمكانية إظهار/إخفاء كل متوسط من الواجهة

### 🎯 نقاط المحور (Pivot Points)
ثلاث طرق لحساب نقاط المحور:
- **Standard** - الطريقة القياسية
- **Fibonacci** - طريقة فيبوناتشي
- **Camarilla** - طريقة كاماريلا

تتضمن:
- R1, R2, R3 (مستويات المقاومة)
- S1, S2, S3 (مستويات الدعم)
- خطوط مرجعية على الرسم البياني

### 📊 مستويات الدعم والمقاومة
- تحديد تلقائي للقيعان والقمم المحلية
- حساب قوة كل مستوى
- عرض مرئي واضح مع ترتيب حسب القوة

### 📉 المؤشرات الفنية الإضافية
- **RSI** - مؤشر القوة النسبية (14 فترة)
- **MACD** - تقارب وتباعد المتوسطات المتحركة
- **Histogram** - رسم بياني للزخم

### ⚡ إشارات التداول التلقائية
- إشارات بيع وشراء ذكية
- تحديد قوة الإشارة (قوية/متوسطة/ضعيفة)
- شرح مفصل لسبب كل إشارة
- تشمل:
  - تقاطع المتوسطات (Golden Cross / Death Cross)
  - تقاطع EMA
  - مستويات المحور
  - مستويات الدعم والمقاومة

## 🛠️ التقنيات المستخدمة

- **Next.js 16** - إطار عمل React
- **TypeScript** - للكتابة الآمنة
- **Tailwind CSS 4** - للتصميم
- **Recharts** - للرسوم البيانية
- **Framer Motion** - للحركات السلسة
- **shadcn/ui** - مكونات UI

## 📦 التثبيت

# الدخول للمجلد
cd trading-indicator-pro

# تثبيت المتطلبات
